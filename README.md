# PetWatch, connececting owners, finders, and local helpers to make reuniting pets faster and easier.

# How To Run:
1. Clone the repository
2. Open the project in an IDE of your choice (PHPStorm recommended)
3. Run setup.php (found in database)
4. Start the local server and have fun!

# Key Features:
- Livesearch: Helps find a specific sighting in the view sighting page alongside radio buttons to filter results
- Map and Card view of sightings: Makes it easier to view sightings in either format
- Photo uploads for reporting sightings 
- Add your own pets: Allows registered users to add their pets to the website and even mark them as lost/found

# Why?
A academic project focused on understanding the client side of web development using CRUD and MVC techniques
